## CNN project - tomato classification
<br>

### 개요
익지 않은 토마토에 존재하는 독성과, 토마토를 재배할 때 드는 노동을 간편화하기 위해 이 프로젝트를 진행하였다. 크게 초록색(익지 않은 토마토)와 붉은색(다 익은 토마토)를 분류했으며 그 중간 지점에 있는 토마토를 얼마나 정교하게 분류하는지를 확인했다. 줄기가 있는 토마토와 없는 토마토, 잘린 토마토 등 다양한 토마토 사진을 이용해 CNN 모델을 만들어 분류했다.
